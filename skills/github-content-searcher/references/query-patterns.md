# Query Patterns

Use focused GitHub search queries instead of broad keyword dumps.

## AI Agent

```text
python ai agent language:Python stars:>100 pushed:>2025-01-01
browser automation agent language:Python stars:>100
multi agent framework language:Python stars:>100
```

## RAG

```text
rag framework language:Python stars:>500
retrieval augmented generation language:Python stars:>100 pushed:>2025-01-01
vector database rag language:Python stars:>100
```

## MCP

```text
mcp server language:Python stars:>50
model context protocol server stars:>100
```

## Skills

```text
codex skill SKILL.md stars:>10
claude skill SKILL.md stars:>10
claude code plugin skill
```

## Practical Rule

Start with one clear domain phrase, then add one or two qualifiers. If results are weak, change the domain phrase before adding more filters.
